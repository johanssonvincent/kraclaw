KEY FINDINGS
================================================================================

SILENT FAILURES IDENTIFIED:
- Messages lost when multiple groups active
- Agent hangs during normal timeout operation
- Scheduled task/retry messages never processed
- Orphaned NATS consumers accumulate

PRODUCTION IMPACT:
- Users experience hung agents with no indication why
- Message loss is silent (no errors in logs)
- Orchestrator unable to recover from transient IPC errors
- Long-running deployments degrade as consumers accumulate

ROOT CAUSE:
The NATS JetStream migration introduced durable consumer management.
Critical bugs in consumer naming and lifecycle mean:
1. Consumers conflict across groups
2. Async cleanup is abandoned mid-operation
3. Consumer state is recreated but not properly initialized
